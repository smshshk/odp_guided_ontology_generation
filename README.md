# ODP-Guided Ontology Generation with Large Language Models

This repo contains the experimental materials associated with our study investigating the effect of **Ontology Design Pattern (ODP) guidance on LLM-driven ontology generation**.

The study compares ontologies generated from **Competency Questions (CQs) alone** with ontologies generated using the same CQs alongside a curated library of **Content Ontology Design Patterns**.


## Repository Contents

The repository provides the complete experimental materials used in the study, including the **competency questions** for the three case studies, the curated **Content ODP library**, the **reference ontologies**, and the resulting **baseline and ODP-guided ontologies**.
